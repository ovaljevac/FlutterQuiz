# Flutter Quiz App 🧠✨

A simple Flutter quiz application that helps you learn Flutter basics in a fun way.  
Users can start the quiz, answer multiple-choice questions, and view a results screen with a summary and final score.

---

## ✨ Features

- Start screen with a clean UI
- Multiple-choice quiz questions
- Results screen with:
  - total score (correct / total)
  - per-question summary (question, your answer, correct answer)
- Restart quiz option
